## Deskripsi
Project ini merupakan implementasi tiga algoritma sorting pada data mahasiswa, yaitu:

- Bubble Sort
- Insertion Sort
- Selection Sort

Data mahasiswa terdiri dari:
- Nama
- NPM
- Nilai
- Ranking

Sorting dilakukan berdasarkan **nilai (descending)** dari yang terbesar ke terkecil.

---

## Algoritma yang Digunakan

### Bubble Sort
- Membandingkan elemen bersebelahan
- Menukar posisi jika tidak sesuai urutan
- Proses dilakukan berulang hingga data terurut

### Insertion Sort
- Mengambil satu elemen
- Menyisipkan ke posisi yang tepat di bagian yang sudah terurut

### Selection Sort
- Mencari nilai terbesar
- Menempatkan ke posisi depan
- Diulang sampai semua data terurut
